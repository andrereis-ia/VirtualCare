# VirtualCare
Assistente Virtual para dúvidas relacionadas a dificuldade de entender prescrição de medicamentos e orientações de uso.

# 🎤 Assistente de Voz com Inteligência Artificial

Este projeto implementa um assistente inteligente capaz de interagir com o usuário por meio de gravação ou upload de áudio.

## 🚀 Funcionalidades

- 🎤 Entrada de áudio (upload ou gravação)
- 🧠 Processamento com IA (ChatGPT)
- 🔊 Resposta em áudio (Text-to-Speech)
- 🔄 Conversão automática de áudio (WAV)

## 🧠 Tecnologias Utilizadas

- Python
- Whisper (Speech-to-Text)
- OpenAI GPT (Processamento de linguagem)
- gTTS (Text-to-Speech)
- Google Colab

## 📊 Fluxo do Sistema

1. Entrada de dados
2. Pré-processamento
3. Conversão para texto
4. Processamento com IA
5. Geração da resposta
6. Conversão para áudio
7. Reprodução

## ▶️ Como executar

1. Abra o notebook no Google Colab
2. Execute as células em ordem
3. Escolha o tipo de entrada

---

## 💡 Observação

Para melhor desempenho, recomenda-se o uso de arquivos de áudio no formato WAV.
